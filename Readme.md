# PF2e Character Gallery Extender

Loads stuff from disabled pf2e adventure modules in [Pathfinder Tokens: Character Gallery](https://foundryvtt.com/packages/pf2e-tokens-characters) even if the game you're running is not PF2e.

## Caveats

The stuff loaded from disabled modules do not show in the configure datasheets. That dialog is fed from internal data sources that cannot be accessed from module land. It might be possible to splice it in with sneaky tricks, but it doesn't do that for now.
